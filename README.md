R Programming Tutorial 📊
A beginner-friendly, hands-on tutorial series covering R programming fundamentals — from basic syntax to data structures and visualization. Built as a structured learning path with practical, runnable examples for every concept.
📖 About
This repository is a collection of small, focused R scripts that each demonstrate a single concept. It's designed so a complete beginner can go file by file, run the code, and build a solid foundation in R before moving on to data analysis or statistics.
🎯 Who This Is For

Absolute beginners starting out with R
Students learning R for data science or statistics coursework
Anyone who wants quick, standalone reference scripts for core R concepts

🚀 Getting Started
Prerequisites

R (version 4.0 or higher recommended)
RStudio (recommended IDE)

Installation
bashgit clone https://github.com/Surya20062425/your-repo-name.git
Open the cloned folder in RStudio, then open any .R or .r file to view and run the code.
How to Use This Tutorial

Start with the basics: syntax.r → variable.r → Data_types.R
Move on to operators and logic: Comparission_operators.R → AND-OR_Operator.R → Conditional_statements.R → Nested_if.R
Learn data structures: vectors/ → list.R and related list scripts → matrices/ → Data_Structures/ → Data_Frame/
Explore control flow: loops/
Learn reusable code: functions/
Finish with visualization: Graphics/ → Bar_chart/ → Pie/

Run each script in RStudio (or via Rscript filename.R in the terminal) and read the comments alongside the code for explanations.
